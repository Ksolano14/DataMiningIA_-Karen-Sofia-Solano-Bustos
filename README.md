# DataMiningIA_-Karen-Sofia-Solano-Bustos
# Proyecto Data Mining con IA

## Objetivo
Analizar los factores que influyen en las ventas mensuales de una tienda.

## Datos
Dataset simulado de ventas con 250 registros.
Variables: mes, producto, precio, cantidad, descuento y total de ventas.

## Limpieza
Se eliminaron duplicados, se corrigieron formatos y se calcularon totales.

## Técnicas aplicadas
Regresión lineal para identificar variables influyentes en ventas.

## Resultados
El descuento influye positivamente en las ventas.
Precios altos reducen el volumen de compra.

## Conclusiones
Las estrategias de descuento ayudan a aumentar ingresos.
El precio debe ajustarse según demanda.

## Limitaciones
Datos simulados.
No incluye factores externos como competencia.

## Recomendaciones
Aplicar análisis con datos reales y agregar segmentación de clientes.
